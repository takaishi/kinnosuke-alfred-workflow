## configure

Write to`~/.kinnosuke` following configure:

```
# set path kinnosuke-clocking-cli
export PATH=~/bin:$PATH

export KINNOSUKE_COMPANYCD="XXXXXX"
export KINNOSUKE_LOGINCD="XXXX"
export KINNOSUKE_PASSWORD="XXXXXXXX"
```
